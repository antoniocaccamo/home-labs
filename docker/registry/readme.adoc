= docker registry 

A local docker registry

== credentials
To create credentials for the docker registry we can use the htpasswd command.

.creating credentials
[source, bash]  
----
mkdir -p auth
echo ${DOCKER_REGISTRY_PASSWORD} | htpasswd -Bnbi antonio > auth/htpasswd

cat auth/htpasswd
antonio:$2y$05$/2akEImMm2Y2NOiGrZrZSehHCALxDeS2uVxu1V67GdAeZ0t9cKjou
----
