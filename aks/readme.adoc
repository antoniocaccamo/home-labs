== Setup Azure CLI

[source,bash]
----
export ARM_SUBSCRIPTION_ID=908894ac-....

terraform init
terraform plan ..
terraform apply
----

.Access Azure and ACR
[source, bash]
----
az login
az acr login --name acraktdev001
----



== kubectl

.Get AKS credentials
[source, bash]
----
az aks get-credentials \
    --resource-group rg-home-lab \
    --name aks-home-lab \
    --admin
----
